# All-In-One Tester APK

Egy professzionális Android diagnosztikai alkalmazás Samsung A35 (One UI 8, Android 16) optimalizációval.

## 🛠️ Funkciók (21+ teszt)

- 🌐 Internet elérés ellenőrzés
- 🔍 DNS lookup
- 🌍 Publikus IP cím
- ⏱️ Weboldal válaszidő
- ⚡ Sebességteszt
- 📡 Ping
- 🔓 Port scanner
- 📁 Mappa méretezés
- 🔋 Akku állapot
- 📶 Hálózati állapot
- 💾 RAM használat
- 💿 Tárhely információ
- 📱 Eszköz info
- ☀️ Képernyő fényerő
- 🧠 CPU magok
- 📦 Telepített alkalmazások
- 📱 Mobil hálózat típusa
- 🎯 Szenzorok
- 🖥️ Kijelző felbontás
- 🔒 Biztonsági patch
- ⚠️ Root ellenőrzés

## 📦 Telepítés

1. Töltsd le az APK-t a GitHub Actions "Artifacts" részéből
2. Engedélyezd az "Ismeretlen forrásból" telepítést
3. Telepítsd a telefonodra

./gradlew assembleDebug
